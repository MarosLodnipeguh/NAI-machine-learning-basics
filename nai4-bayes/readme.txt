Zawarty w tym folderze data set stanowi modyfikację przykładu z zajęć dotyczącego pogody.
Poniżej przedstawione są atrybuty oraz ich możliwe wartości:
1. Zachmurzenie/opady - "slonecznie", "zachmurzenie", "deszcz", "snieg".
2. Wiatr - "tak", "nie".
3. Wilgotność - "wysoka", "srednia", "niska".
4. Temperatura - "wysoka", "srednia", "niska".
5. Atrybut decyzyjny czy wyjść z domu (tylko dla zbioru treningowego) - "tak", "nie".
UWAGA! Program ma sam zliczać atrybuty oraz ich możliwe wartości. Ich powyższe wylistowanie służy tylko i wyłącznie do ułatwienia testowania programu. Proszę pamiętać również, że atrybut decyzyjny także może przyjąć więcej niż 2 wartości.
